# Casting contract

- Canonical source: the checked-out repository root.
- Select exactly one `projects/<project-id>/` and its declared production target.
- Convert inspiration to a Reference Brief, write a Coverage Report and validate its route before writing a Manifest.
- Use only Registry entries whose status is `Approved` and that include a component key.
- Require Library Distribution and Manifest Preflight before any Figma write.
- Page container Frames must be Manifest-declared allowed literals.
- Missing visual/API assets produce a Mold Gap; unpublished Library, permission and font issues produce a Gap Report; content overflow returns to Content Plan. All stop the current object, but only Mold Gap enters Mold Making.
